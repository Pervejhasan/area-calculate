DOM-Document Object Model . We can access, update, add and delete every html element, content of element, attribute of element by dom.

we can select element from dom different ways
1. select by id
2. select by className
3. select by tagName
4. select by addEventListener

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event than it search upper elements where any event set then all event trigger simultaneously. If we want than we can stop Event Bubbling by using stopPropagation() method.

If a function pass as a parameter another function than call the parameter function inside the parent function than we called that parameter function callback function.
